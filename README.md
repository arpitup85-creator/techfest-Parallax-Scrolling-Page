# Parallax Scrolling Page

A simple webpage built for Techfest CA Program using HTML, CSS, and TypeScript.

##   How to Run
1. Clone the repo in vscode editor:
   ```bash
   git clone https://github.com/arpitup85-creator/techfest-Parallax-Scrolling-Page.git

2. Run with Vite in terminal (Recommended):
    # Install dependencies:
     npm install

    # Start the dev server:
     npm run dev

3. Open the local server link (usually http://localhost:5173) in your browser
